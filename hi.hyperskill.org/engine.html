Our features
Knowledge map
an interface for creating interconnected concept maps. For each concept you can add textual explanation and set clear prerequisite a student needs to understand in order to grasp it.
Extensive in-browser practice
add practical quizzes to each concept including multiple choice, matching, open answers and coding assignments.
Curriculum
create paths for your students from one point A of the concept map point B. Curriculum will offer students a linear structure of the material with various activities that can prompt better learning.
