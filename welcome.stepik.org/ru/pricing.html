Что такое приватный курс?
Приватный курс — это закрытый курс на Stepik, доступ к которому определяет автор.
Курс не отображается по прямой ссылке, а также в каталоге Stepik.
Автор сам назначает доступ для своих учащихся, в том числе на платной основе.
Кому и для чего могут пригодиться
приватные курсы?
Компаниям:
  • Для обучения сотрудников, партнеров, клиентов.
  • Для тестирования, в том числе соискателей на вакансии.
  • Для реализации перевернутого обучения (теория — онлайн, практика — очно).
Учебным центрам:
  • В качестве дополнения к очным программам для выравнивания знаний учащихся.
  • Для перевода очного обучения в онлайн формат.
Образовательным учреждениям:
  • Для обучения студентов, очных и заочных форм.
  • Для организации программ дополнительного профессионального образования.
  • Для проведения внутренних олимпиад, конкурсов.
Преподавателям:
  • Для перевода из очного обучения в онлайн, чтобы увеличить количество учащихся.
  • Для проверки домашних заданий онлайн и автоматизации проведения тестирования.
  • Для автоматизации проверки заданий на программирование.
Бизнес-тренерам:
  • Для создания и проведения онлайн-курса в дополнение к очным программам.
  • Для организации платного дистанционного обучения.
Как создать приватный курс?
  • Пройдите регистрацию на Stepik.
  • Перейдите в раздел Создание курса.
  • Измените доступ к курсу на приватный.
  • Следуйте рекомендациям по оформлению курса.
  • Пригласите ваших студентов к прохождению курса
Преимущества приватных курсов на Stepik

Создавайте закрытые материалы для вашей аудитории.
Продавайте доступ к курсу.
Встроенный редактор курсов
В стоимость подписки входит редактор онлайн-курсов. Добавляйте в курс видео, файлы, текстовые и практические шаги.
Нет ограничения на число студентов
Обучайте любое количество студентов на вашем курсе. С ростом числа учащихся стоимость подписки не изменится.
Общение со слушателями
Общайтесь в комментариях под каждым шагом в курсе, создавайте форумы для совместного обсуждения.
Создавайте автоматические рассылки.
Практические задания
Используйте тестовые задания, а также задачи на программирование с автоматической проверкой и кросс-рецезированием в группе. Проводите экзамены с прокторингом (сервис верификации личности).
Быстрое подключение оплаты
Настройте оплату через сервис TimePad без дополнительных настроек. Stepik не берет никаких дополнительных комиссий с проданных курсов.
Если у вас настроен прием платежей на сайте, используйте API Stepik для предоставления доступа к курсам.
Электронные сертификаты
Вы можете настроить автоматическую или разовую выдачу электронных сертификатов. Настройте оформление и условия генерации.
Учебная аналитика
Работайте с отчетами и статистикой по курсу, следите за успеваемостью студентов. Используйте автоматическую аналитику качества курса.
Мобильные приложения
Бесплатные приложения для iOS- и Android-утройств с возможностью прохождения задач на программирование и сохранения видео в оффлайн-режим.
